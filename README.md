# 儒释道学习

## 介绍

儒、释、道三家典籍学习记录摘要。  
同时发心整理三家经典文集，使其存在于网络之中，并能自由下载和修改完善，**欢迎大家参与**。  
文件使用markdown格式，在方便阅读的同时也能及时修改其中抄录错误。并可以以代码形式加以追踪回溯。  

## 初步约定  

1.文件统一采用utf-8编码格式；  
2.文件扩展名为md；  
3.内部书名为标题一（ \#加空格），其余标题递减。  

如有更好想法，欢迎提交[issue](https://github.com/sxmwhl/Confucianism-Buddhism-Taoism/issues)讨论。
